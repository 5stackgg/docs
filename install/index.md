# Installation

The [5Stack-Panel](https://github.com/5stackgg/5stack-panel) repository includes scripts to help facilitate the installation process.

```bash
git clone https://github.com/5stackgg/5stack-panel; cd 5stack-panel; ./install.sh
```

::: danger
Please ensure you have met the [Requirements](/install/requirements) before you begin installing the 5Stack Panel.
:::

## Running the Installation Script

Inside the 5Stack-Panel directory, run the installation script `./install.sh`.

After installing, you can make changes to the configuration files in the `overlays/config` or `overlays/local-secrets` directories. If modified, you will need to update 5Stack. See the [Updating Guide](/updating) for instructions.
